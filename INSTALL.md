# BreakReminder — Install on a new Mac

A tiny menu-bar app that dims your screen every 20 minutes to remind you to rest your eyes and stand up.

---

## Safety & privacy (please read)

This app was generated by **Claude** (Anthropic's AI coding assistant) and built locally on your Mac from the included source code. You can read every line yourself in `BreakReminder.swift` (~300 lines of Swift).

**What this app does NOT do:**
- ❌ Does not connect to the internet (no network access, no telemetry, no analytics)
- ❌ Does not collect, upload, or store any personal data
- ❌ Does not access your camera, microphone, files, contacts, or clipboard
- ❌ Does not monitor keystrokes outside its own break-reminder overlay window
- ❌ Does not run at elevated / admin privileges
- ❌ Does not auto-update or download anything after install

**What this app DOES do:**
- ✅ Shows a menu-bar icon with a countdown (`☕ 19:45`)
- ✅ Dims your screen with a full-screen overlay every 20 minutes (dismissible at any time)
- ✅ Listens to macOS's public screen-lock notification so it can reset after long rests
- ✅ Uses only standard Apple AppKit APIs — no third-party libraries, no dependencies

**One thing to note:** This app is **not signed with an Apple Developer certificate**, so the first time you open it macOS will warn you that it's from an "unidentified developer". This is normal for personal / non-commercial apps — it doesn't mean the app is malicious, just that Apple hasn't charged $99/year to certify it. See Step 4 below for how to allow it.

Provided **as-is**, without warranty. Delete any time — see the **Uninstall** section at the bottom.

## What you need

- macOS 11 (Big Sur) or newer
- **Xcode Command Line Tools** (free, ~1 GB). If `swiftc` is already installed, skip this.
  ```bash
  xcode-select --install
  ```
  Click "Install" in the popup. Wait 5–15 minutes.

You do **not** need the full Xcode app, Homebrew, or any other dependency.

## Step-by-step install

### 1. Get the folder
Download the `break-reminder` folder (from Google Drive, USB, email, etc.) and put it wherever you like, e.g.:

```
~/Documents/break-reminder/
```

The folder should contain at least:
- `BreakReminder.swift` — the source code
- `build.sh` — the build script
- `INSTALL.md` — this file

### 2. Build it
Open Terminal and run:

```bash
cd ~/Documents/break-reminder
./build.sh
```

You should see `Build complete.` and a new `BreakReminder.app` appear in the folder.

If you see "permission denied", run:
```bash
chmod +x build.sh
./build.sh
```

If you see `swiftc: command not found`, install Xcode Command Line Tools (see "What you need" above) and try again.

### 3. Run it
```bash
open BreakReminder.app
```

Look at the top-right of your menu bar — you should see `☕ 20:00` counting down every second.

### 4. First-launch security warning (likely)
Because the app is not signed by Apple, macOS Gatekeeper may block it:

> *"BreakReminder" cannot be opened because it is from an unidentified developer.*

Fix it **once**:
- Right-click (or Control-click) `BreakReminder.app` in Finder → **Open** → click **Open** in the popup.
- Or: System Settings → Privacy & Security → scroll down → click **Open Anyway** next to the BreakReminder warning.

After that, it will launch normally every time.

### 5. Auto-start at login (recommended)

First, move the app to the standard `/Applications` folder so it shows up in the Login Items picker. In Terminal:

```bash
mv ~/Documents/break-reminder/BreakReminder.app /Applications/
```

Then add it to Login Items:
1. Open **System Settings**
2. Go to **General → Login Items & Extensions** (on older macOS: **General → Login Items**)
3. Under "Open at Login", click the **+** button
4. Pick **BreakReminder** from the list (in `/Applications`)
5. Click **Open** / **Add**

You should see `BreakReminder` in the list. Next time you log in, it starts automatically.

To disable later: select it in the list and click the **–** button.

> After moving to `/Applications`, if you ever want to update the app (edit `BreakReminder.swift`), just run `./build.sh` again — it automatically syncs the copy in `/Applications`.

## How to use

Click the `☕ MM:SS` icon in the menu bar:

| Menu item         | What it does                                             |
| ----------------- | -------------------------------------------------------- |
| Next break in …   | Live countdown to the next break                         |
| Pause / Resume    | Stop the timer / continue                                |
| Take Break Now    | Trigger the overlay immediately (great for testing)      |
| Reset Timer       | Restart the countdown from full                          |
| Interval ▸        | 5 / 10 / 15 / 20 / 30 / 45 / 60 minutes                  |
| Quit              | Close the app                                            |

**During a break**, press any of: **Esc**, **Space**, **Enter**, **click anywhere**, or click the red **Skip** button. The overlay also auto-closes after 20 seconds on its own.

**Lock-based reset**: If you lock your screen (`Ctrl+Cmd+Q`) for **5 minutes or more**, the timer automatically resets when you unlock — because you already got a rest.

## Uninstall

1. Remove `BreakReminder` from **System Settings → General → Login Items**.
2. Delete the app and source folder:
   ```bash
   rm -rf /Applications/BreakReminder.app
   rm -rf ~/Documents/break-reminder
   ```

## Troubleshooting

**`swiftc: command not found`**
Run `xcode-select --install` and wait for it to finish.

**`./build.sh: permission denied`**
Run `chmod +x build.sh` first.

**App launches but no menu-bar icon appears**
The menu bar might be full. Hide some other icons, or try: `killall BreakReminder && open BreakReminder.app`.

**Overlay won't close**
Ctrl+Cmd+Q to lock, or in Terminal: `pkill -f BreakReminder`.

**Want to change the break duration, interval defaults, colors, messages?**
Edit `BreakReminder.swift`, then re-run `./build.sh`.
